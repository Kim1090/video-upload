# README #

### What is this repository for? ###

This project provide service upload video and capture image preview

### How do I get set up? ###
```bash
	git clone https://Kim1090@bitbucket.org/Kim1090/video-upload.git
```

```bash
	cd video-upload
	npm install
```

```bash
	cp config.template.json config.json
```

```bash
	pm2 start server.js
```
